# COVID-19 Model Inventory and Archive

## COVID Act Now (covidactnow.org) model

This subdirectory contains state-level data produced and made publicly available by the COVIDactnow.org website.

Additional details about this model and its underlying data and assumptions can be found on the [COVID Act Now website](https://covidactnow.org/). Data are updated daily via the COVID Act Now website and will regularly be pulled and archived by this model inventory. Data were last pulled and updated in this repository on April 10, 2020.
